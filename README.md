# 书单

在时间轴上记录每年所读的书

## 使用工具

 - [beego](https://beego.me)

 - sqlite

 - [时间轴模板](http://demo.lanrenzhijia.com/2013/js901204/)

 ## 实现功能

  - [x] 时间轴展示一年内所有书及简介

  - [] 后台添加书籍基本信息

  - [] 读书思维导图（[百度脑图](http://naotu.baidu.com)）

  - [] 读书笔记(富文本)

  - [] 读书摘要（[kindle剪切板](https://bookfere.com/post/110.html)）
